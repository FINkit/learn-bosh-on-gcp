The BOSH Bastion is the jumpbox we will use to access the BOSH Director, and the BOSH Environment in general. It is created using Terraform.

To create a BOSH Bastion, clone the bosh-on-gcp repository, move the service account key into the bosh-on-gcp directory and execute the build-bosh-resources.sh script:

$ git clone https://github.com/finkit/bosh-on-gcp

$ mv gcp-automated-user.key.json bosh-on-gcp/gcp-automated-user.key.json

$ cd bosh-on-gcp/terraform

$ ./build-bosh-resources.sh -p <YOUR_GCP_PROJECT_NAME>

Should the build-bosh-resources.sh script fail with an error indicating that the region required field is not set, use the command "gcloud config set compute/region <YOUR_GCP_PROJECT_REGION>" (where <YOUR_GCP_PROJECT_REGION> is the region for your project, e.g. europe-west2).

Should the build-bosh-resources.sh script fail with an error loading the zone, use the command "gcloud config set compute/zone <YOUR_GCP_PROJECT_ZONE>" (where <YOUR_GCP_PROJECT_ZONE> is the zone for your project, e.g. europe-west2-a).
