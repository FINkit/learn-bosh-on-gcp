The BOSH Bastion is the jumpbox we will use to access the BOSH Director, and the BOSH Environment in general. It is created using the Google Cloud SDK.

To install the Google Cloud SDK follow the instructions on the Cloud SDK page.

Once the Cloud SDK is installed, initialize a gcloud session:

$ gcloud init

To create the BOSH Bastion, clone the bosh-on-gcp repository and execute the create-project.sh script:

$ git clone https://github.com/finkit/bosh-on-gcp

$ cd bosh-on-gcp

$ ./create-project.sh <GCP_PROJECT>
