The BOSH Bastion is the jumpbox we will use to access the BOSH Director, and the BOSH Environment in general. It is created using the Google Cloud SDK and Terraform. To install the Google Cloud SDK follow the instructions on the Cloud SDK page.

Once the Cloud SDK is installed, initialize a gcloud session:

$ gcloud init

To create a GCP Project and BOSH Bastion, clone the bosh-on-gcp repository and execute the create-project.sh and build-bosh-resources.sh scripts:

$ git clone https://github.com/finkit/bosh-on-gcp

$ cd bosh-on-gcp

$ ./create-project.sh -p <GCP_PROJECT>

$ cd terraform

$ ./build-bosh-resources.sh -p <GCP_PROJECT>

Should the build-bosh-resources.sh script fail with an error indicating that the region required field is not set, use the command "gcloud config set compute/region <GCP_PROJECT_REGION>" (where <GCP_PROJECT_REGION> is the region for your project, e.g. europe-west2).

Should the build-bosh-resources.sh script fail with an error loading the zone, use the command "gcloud config set compute/zone <GCP_PROJECT_ZONE>" (where <GCP_PROJECT_ZONE> is the zone for your project, e.g. europe-west2-a).
