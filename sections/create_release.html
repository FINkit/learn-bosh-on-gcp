We are going to use a simple BOSH release that deploys an http server.

$ git clone https://github.com/finkit/learn-bosh-on-gcp-release

$ cd learn-bosh-on-gcp-release

$ bosh create-release

Upload generated release to Director:

$ bosh upload-release

Check uploaded releases:

$ bosh releases

Using environment '192.168.50.6' as client 'admin' Name Version Commit Hash learn-bosh-on-gcp 0+dev.1 fd90933 1 releases
